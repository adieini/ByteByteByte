# StarVest 🌾🛰️

Built at DA Hacks 2024

StarVest is a satellite-driven farming assistant app that helps farmers make smarter, data-driven decisions. It combines real-time weather and soil moisture data with user-inputted crop and location info to provide tailored recommendations on planting, harvesting, and disease risk.

## Features
- 📍 Location-based insights using Google Maps API
- 🌦️ Real-time satellite weather and soil data
- 🤖 Machine learning model for personalized crop recommendations
- 📱 Cross-platform mobile app (iOS & Android) built with React Native and Expo

## Tech Stack
React Native, TypeScript, JavaScript, Expo, Google Maps API, Machine Learning
